    업데이트 내역

    2024.01.25

    • 🚀 새 모델 출시: GPT-4 Turbo 프리뷰, GPT-3.5 Turbo 업데이트, 새로운 텍스트 모더레이션 모델 포함
    • 🆕 GPT-4 Turbo 업데이트: 새로운 GPT-4 Turbo 프리뷰 모델(gpt-4-0125-preview) 출시로 코드 생성 등의 작업을 더 철저히 완료하고, 비영어 UTF-8 생성에 영향을 주는 버그 수정
    • 📈 70% 이상의 GPT-4 API 사용자가 업데이트된 지식 컷오프, 더 큰 128k 컨텍스트 윈도우, 낮은 가격의 이점으로 GPT-4 Turbo로 전환
    • 🔄 자동 업그레이드: 새로운 GPT-4 Turbo 프리뷰 버전으로 자동 업그레이드하길 원하는 사용자를 위해 gpt-4-turbo-preview 모델명 별칭 도입
    • 🔜 GPT-4 Turbo with Vision 출시 예정: 몇 달 내 일반 사용자를 위한 GPT-4 Turbo with Vision 출시 계획
    • 🛡️ 모더레이션 모델 업데이트: 개발자가 잠재적으로 해로운 텍스트를 식별할 수 있도록 하는 무료 Moderation API의 가장 강력한 모더레이션 모델인 text-moderation-007 출시
    • 💡 임베딩 모델 개선: text-embedding-3-small과 text-embedding-3-large 출시로 성능 향상 및 가격 인하
      • 📉 text-embedding-3-small은 가격이 $0.0001에서 $0.00002로 5배 인하
      • 🌟 text-embedding-3-large는 최고 성능을 제공하며 가격은 $0.00013 / 1k 토큰
    • 🛠 GPT-3.5 Turbo 가격 인하: 입력 가격 50% 인하($0.0005 / 1K 토큰), 출력 가격 25% 인하($0.0015 / 1K 토큰)
    • 📊 API 키 관리 및 사용량 이해 개선: 개발자가 API 키에 대한 권한을 할당하고 사용량을 더 잘 이해할 수 있도록 플랫폼 개선
    • 🔒 데이터 보호 강화: 기본적으로 OpenAI API로 전송된 데이터는 모델 훈련이나 개선에 사용되지 않음
